# Fala Cidadão

Plataforma cívica (MVP) para registrar demandas urbanas com foto, geolocalização e rastreabilidade.
Protótipo desenvolvido para a trilha/projeto de inovação (CESAR School).

## Objetivo
Facilitar que cidadãos registrem problemas urbanos (ex.: buracos, iluminação, lixo, esgoto) com evidências e localização, permitindo acompanhamento do status e transparência do processo.

## Principais funcionalidades (MVP)
- Registro de demanda com categoria, descrição e evidências (fotos)
- Temas por cidade (Recife / Olinda / Jaboatão)
- Painel público de demandas (visualização e consulta)
- Persistência e simulação de dados via LocalStorage (sem backend)

## Tecnologias
- React + Vite
- Tailwind CSS
